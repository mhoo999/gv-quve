# QUVE 브랜드 가이드라인

VERSION 1.0 (updated 2025.09.03)

## 브랜드 소개

**Quve(큐브)**는 3~7세 유아동을 대상의 AI 기반 인터랙티브 콘텐츠 서비스입니다.

기존의 일방향적인 영상 시청이 초래하는 사고력 저하 및 콘텐츠 중독과 같은 부작용을 해결하고자 기획되었으며, 아이들이 영상 속 AI 캐릭터와 질문과 응답을 주고받으며 능동적으로 콘텐츠에 참여할 수 있도록 설계되어 있습니다.

영상 중간에 AI가 질문을 던지고, 아이는 이를 생각하고 답함으로써 사고력, 표현력, 창의력을 자연스럽게 키워나갈 수 있으며, 즐거운 상호작용 경험을 통해 아이들은 단순 시청을 넘어선 깊이 있는 인지 발달을 이룰 수 있는 브랜드입니다.

## 타이포그래피 (Typography)

### 영문 서체: FUTURA

QUVE의 영문 지정 서체입니다. 둥근 박스형의 조형 안에서 느껴지는 형태감이 QUVE의 브랜드 아이덴티티에 즐거움과 현대적인 표현으로 전달합니다.

**사용 가능한 웨이트:**
- FUTURA Regular
- FUTURA Medium
- FUTURA Bold

**사용 예시:**
```
QUVE
Brighter Growing Moment
```

### 국문 서체: Pretendard

QUVE의 국문 지정 서체입니다. 현대적이고 가독성 측면에서 정확한 느낌을 전달하는 Pretendard는 QUVE의 브랜드 아이덴티티를 효과적으로 전달합니다.

**다운로드:** https://cactus.tistory.com/306

**사용 가능한 웨이트:**
- Pretendard Regular
- Pretendard Medium
- Pretendard SemiBold

**사용 예시:**
```
유아동 언어 발달 솔루션 QUVE
성장으로 이어지는 더 빛나는 순간
```

## 컬러 팔레트 (Color Palette)

QUVE 브랜드를 표현하는 색상입니다. 브랜드 아이덴티티를 전달하는 가장 핵심적인 요소로 색상값을 참고하여 매체에 적용하는 것을 권장합니다.

### QUVE Violet (메인 컬러)
- **HEX:** `#a402f9`
- **RGB:** R164 G2 B249
- **CMYK:** C54 M81 Y0 K0
- **용도:** 브랜드 메인 컬러, 로고, CTA 버튼, 강조 요소

### QUVE Pink (서브 컬러)
- **HEX:** `#ff3be2`
- **RGB:** R250 G80 B220
- **CMYK:** C11 M82 Y0 K0
- **용도:** 그라데이션 효과, 포인트 컬러

### QUVE Yellow (액센트 컬러)
- **HEX:** `#fab400`
- **RGB:** R250 G180 B0
- **CMYK:** C0 M34 Y100 K0
- **용도:** 강조 텍스트, CTA 버튼, 하이라이트

### QUVE Deep Violet (다크 컬러)
- **HEX:** `#4d1696`
- **RGB:** R77 G22 B150
- **CMYK:** C82 M100 Y0 K0
- **용도:** 배경, 그라데이션 베이스, 어두운 영역

### QUVE White
- **HEX:** `#ffffff`
- **용도:** 배경, 텍스트 (다크 배경 위)

## 로고 사용 가이드

### 로고 최소 여백 (Clearspace)
로고 주변에 최소 **1X** (로고 높이 기준) 만큼의 여백을 확보합니다.

### 로고 최소 크기
- 최소 높이: **20px**

### 로고 사용 원칙

#### 권장 사용법
1. **On White:** 흰색 배경 위에 메인 컬러 로고
2. **On Black:** 검은색 배경 위에 화이트 로고
3. **On Color:** 브랜드 컬러 배경 위에 화이트 로고
4. **Color On Black:** 검은색 배경 위에 컬러 로고

#### 사용 금지 사항
- 브랜드 로고의 비례를 임의로 조정하여 사용 금지
- 브랜드 로고의 조합 간격을 임의로 변경 금지
- 브랜드 로고의 형태나 굵기를 임의로 변형 금지
- 브랜드 로고 조합을 임의로 변경 금지
- 브랜드 로고의 색상을 임의로 변형 금지
- 브랜드 로고의 디테일을 임의로 변경하거나 삭제 금지
- 브랜드 로고에 테두리 효과 적용 금지
- 가독성이 저해되는 배경 색상/이미지에서 사용 금지

## 그래픽 모티프 (Brand Motif)

QUVE의 특징을 살린 그래픽 모티프입니다.

### A. 빛나는 그래픽 요소
심볼의 조형에서 직관적 사용하는 그래픽 가이드입니다. 빛나는 그래픽 요소가 보이는 선에서 크롭하여 사용 권장합니다.

### B. 쉼표 형태의 원형 부분
쉼표 형태의 원형부분이 보이는 선에서 크롭하여 사용 권장합니다.
